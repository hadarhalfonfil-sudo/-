# 🌈 דף קשר - מערכת עדכוני גן

מערכת דיגיטלית פשוטה ליצירת דפי עדכון מעוצבים להורים של ילדי הגן.

## 📱 קישור למערכת

לאחר העלאה ל-GitHub Pages, הקישור יהיה:
```
https://[שם-המשתמש].github.io/[שם-הפרויקט]/
```

## 🚀 איך להעלות ל-GitHub?

### שלב 1: יצירת חשבון
1. גשו ל-https://github.com
2. לחצו על **Sign up**
3. הירשמו עם אימייל וסיסמה

### שלב 2: יצירת Repository חדש
1. לחצו על ה-**+** למעלה מימין
2. בחרו **New repository**
3. מלאו:
   - **Repository name**: `kindergarten` (או כל שם שתרצו)
   - ✅ סמנו **Public**
   - ✅ סמנו **Add a README file**
4. לחצו **Create repository**

### שלב 3: העלאת הקובץ
1. בדף המאגר, לחצו **Add file** → **Upload files**
2. גררו את הקובץ `index.html` לתוך הדף
3. לחצו **Commit changes**

### שלב 4: הפעלת GitHub Pages
1. לחצו **Settings** (בתפריט למעלה)
2. לחצו **Pages** (בתפריט השמאלי)
3. תחת **Source**:
   - Branch: **main**
   - Folder: **/ (root)**
4. לחצו **Save**
5. המתינו 2-3 דקות
6. רעננו את הדף - תראו את הקישור שלכם! 🎉

## 🎯 איך להשתמש?

1. פתחו את הקישור שלכם
2. מלאו את השדות הרלוונטיים (כל שדה אופציונלי!)
3. לחצו "צרו מסמך מעוצב"
4. לחצו "הדפסה ושמירה כ-PDF"
5. בחרו "שמירה כ-PDF" במדפסת
6. שלחו את הקובץ להורים בוואטסאפ! 💬
